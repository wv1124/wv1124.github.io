X30打印机设置

X30目前支持两个厂家的：

厂商 型号 说明
上海湃达 PD-X30便携式蓝牙标签打印机
厦门捷昕 X30便携式票据打印机

  1. 湃达PD-X30便携式蓝牙标签打印机。

  2. 厦门捷昕智能技术有限公司的便携式票据打印机。

目前云小店没有设置打印机打印方式，直扫通过打印机名称去判断是标签打印，还是小票打印机(后面版本会改成先连接再设置打印机属性，用户去设置是标签打印机还是小票打印机，是打印80mm还是58mm)。

打印纸: 75mm*40mm标签打印纸，各家纸标签之前间隔有少许出入，可能产生打一张跳一张问题，下面有解详细说明解决方法。

1. 打印机键位

键位: 从左到右依次: 设置键, 走纸键, 电源键

2. X30打印机设置

2.1 初级设置：

进入方式: 长按设置键3秒, 出现菜单, 再按设置键切换菜单, 按走纸键切换选项, 按电源键保存设置

3. 价签打印机跳纸问题

问题描述：商品标签打印，隔一个空白不打印。
键位: 从左到右依次: 设置键, 走纸键, 电源键

  1. 同时长按设置键和走纸键3秒左右，出现菜单。
  2. 按左侧设置键切换菜单到：退纸, 按打印键切换11毫米。
  3. 按电源键保存设置。
  4. 测试打印。
  5. 问题未解决重复1的操作，退纸设置为10毫米或者12毫米，再测试。

4. 打印机常用指令集

目前主流的打印机控制语言只要有三种：
Epson公司的ESC命令集（普通打印机）
HP公司的PCL命令集
Adobe公司的PostScript(简称PS)命令集
其它：CPCL命令集（移动打印机）, TSPL命令集(标签打印机)

4.1 ESC指令

Epson公司的ESC命令集（普通打印机）。
58mm和80mm小票打印都用这种方式打印。

4.2 CPCL指令

CPCL命令集文档地址：http://www.docin.com/p-2160105026.html
可以指定位置打印，X30就是CPCL指令集。
支持图打。

这就是我们为什么选择CPCL指令原因。

### 4.3 TSPL指令
跟ESC指令类型，无法指定位置打印，无法居中打印。
支持图打。
不能以文字模式打印如下标签：

只能用图片方式打印，打印比较慢。
