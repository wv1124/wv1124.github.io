2. Charles

Mac下前端开发必备神器，愿意会付钱购买的好工具。

2.1 浏览器基本抓包

2.3 Map Local

把一个请求转到本直文件，请求会直接返回本地文件。应用场景，PC线上页面调试。

2.4 Mirror

把指定条件请求和响应保存到本地文件系统。可以先把某些请求全部缓存到本地，实现离线调试页面。
