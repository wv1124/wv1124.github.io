大华条码秤

大华条码秤

云小店Android版支持大华TM-15F系列和TM-30Ab系列条码电子秤，通过收银机同步商品到大华电子秤，电子秤称商品打印条码标签，收银机扫描条码标签识别商品和总价，最后通过收银机下单和支付完成交易流程。

如果使用其它收银同步商品后可能会出现条码标签不能识别问题，请按下面【初次使用云小店收银设置条码秤】操作设置，才能正常使用条码秤。

目前用户主要反馈两个问题：

  1. 联网问题。参考网络设置中设置IP
  2. 条码格式问题。参考【初次使用云小店收银设置条码秤】解决。

初次使用云小店收银设置条码秤

  1. 电子秤恢复出厂设置：功能 => 9371 => 确认 => 清除之前的设置信息(恢复电子秤默认设置)，这步操作主要是重置条码格式 为: FFWWWWWEEEEEC。
  2. 设置条码格式: 设置 => 39706 => 确认 => 按去皮4次=> 条码类型: 05 => 确认。 05对应的条码格式：FFWWWWWEEEEEC，这也是大华默认条码格式。
  3. 修改店名为20: 设置 => 39706 => 确认 => 按去皮26次=>店名: 20 => 确认。
  4. 云小店收银同步商品。

大华条码打印出来不是2开始的13位数字

请按上上面【初次使用云小店收银设置条码秤】重新设置电子秤。

如何设置条码秤IP

通过路由器连接收银机和条码秤，收银机不需要特殊设置，可以使用自动获取IP。条码秤需要设置为一个局域网中未分配IP地址，请按下面操作：

  1. 获取收银机IP:
    无线连接时, Android系统 设置 => WLAN => 右上角设置图标 => IP地址，如192.168.0.10, 这样的就是IP地址。
    有线连接时, Android系统 设置 => 以太网 => IP地址，如192.168.0.10, 这样的就是IP地址。

  2. 获取局域网中未使用的IP地址:
    打开云小店 => 左上角图标 => 设置 => 硬件设置 => 标签称重机 => 选中一个按键模板(如果没有就新建一个并选中) => 右侧输入IP地址中输入一个比上面IP地址大的地址，如:192.168.2.11 => 检测 => 提示连接失败，说明没此IP地址未被使用，接着进行下一步，如果连接成功，请再增加最后一位直到提示连接失败。

  3. 设置条码秤IP:
    使用上一步获取的未使用的IP地址，设置条码秤IP，如获取到未使用IP地址是192.168.2.11, 设置如下: 功能 => 9002 => 确认 => 192 => 去皮 => 168 => 去皮 => 2 => 去皮 => 11 => 确认。完成设置稍等5秒左

  4. 收银机检查条码秤IP:
    如上一步设置条码秤IP为192.168.2.11，就可以云小店中标签称重机界面右侧输入IP地输入框中输入 192.168.2.11 检测上一步设置IP地址是否正常。如果提示连接成功就可以同步商品了。如果提示连接失败，就需要检查条码秤和路由器线路连接了。

常用设置

  1. 设置IP, 如设置IP为: 192.168.0.150
    功能 => 9002 => 确认 => 192 => 去皮 => 168 => 去皮 => 0 => 去皮 => 150 => 确认

  2. 清除设置
    功能 => 9371 => 确认 => 清除之前的设置信息(恢复电子秤默认设置)

  3. 修改店名
    设置 => 39706 => 确认 => 按去皮26次=>店名: 20 => 确认
